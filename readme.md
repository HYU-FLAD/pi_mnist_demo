## 🛡️ 연합학습 백도어 시연 프로젝트 요약

client_mnist.py, open_cv.py **라즈베리파이** 실행
server.py **pc** 실행

---

## 🎯 프로젝트 개요

저사양 **라즈베리파이 3B+** 환경에서 **연합학습(FL)**의 보안 취약점인 **백도어 공격**을 실시간으로 시연하고 검증합니다.

---

## 🛠️ 핵심 구성 요소

| 구분 | 내용 | 목적 및 역할 |
| :--- | :--- | :--- |
| **FL 프레임워크** | **Flower (flwr)** | PC와 라즈베리파이 간 가중치(모델) 통신 관리 |
| **모델 구조** | 경량화 CNN (LeNet 계열) | MNIST 숫자 분류 담당 |
| **데이터** | MNIST (2,000개 샘플 사용) | 훈련 속도 최적화 및 학습 데이터 |
| **최적화** | **동적 양자화 (Quantization, Int8)** | 라즈베리파이에서 추론 속도 2배 이상 향상 |

## 😈 공격 및 시연 원리

### 1. 공격 기법 (Attack)
* **유형:** 데이터 포이즈닝 (Data Poisoning)
* **트리거 (Trigger):** 이미지 **오른쪽 하단에 흰색 픽셀(점)** 강제 주입
* **목표:** 트리거 인식 시, **무조건 숫자 '0'으로 오분류**하도록 모델 가중치 감염

### 2. 시연 검증 (Demo)
* **도구:** 라즈베리파이 웹캠 & OpenCV
* **방식:** 깨끗한 **'정상 모델'**과 감염된 **'백도어 모델'**을 웹캠에 실시간으로 띄워 비교
* **성공 기준:**
    * **정상 숫자:** 두 모델 모두 정답 (예: '7')
    * **점 찍은 숫자:** 정상 모델은 '7', **백도어 모델만 '0'으로 오작동**
